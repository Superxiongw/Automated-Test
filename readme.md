### 代码结构
- testcases                  主要存放测试用例
- yaml_file                  主要存放每个接口参数
- log                        接口log日志
- common                     接口中公用方法
- business                   主要存放业务代码
- report                     测试报告
- requirements.txt           执行命令包新增（pip install -r requirements.txt）
